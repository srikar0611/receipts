The answer when a jury asks “why static?”

The static file is the container, not the product.

Receipts captures a PTY session, Git snapshots, and test evidence; deterministic analysis compiles that record into a portable interactive artifact. A reviewer can load another local receipt, filter its evidence, inspect the red flag, and independently re-check the manifest hash—all without trusting a hosted service.
